# 图书信息管理系统 python401

本项目是基于Python和PyQt5开发的图书信息管理系统。该系统提供了一套完整的源代码，包括UI文件及源码，可供学习和参考。

## 技术栈

- **后端**: Python
- **前端**: PyQt5

## 功能模块

以下是系统的核心功能模块：

1. **图书信息显示**: 显示图书表中所有图书的相关信息。
2. **图书信息插入**: 将图书信息插入到指定位置。
3. **图书信息删除**: 删除指定位置的图书。
5. **图书去重**: 实现图书信息的去重功能。
6. **图书模糊查询**: 模糊搜索图书信息。
7. **图书批量修改**: 批量修改图书信息。
9. **登录注册系统**: 管理用户的登录和注册。
10. **权限管理系统**: 对不同用户设置不同的权限。

## 系统角色

系统主要分为以下角色：

- **普通用户**: 可以查询和浏览图书信息。
- **管理员**: 拥有图书信息的增删改查权限，以及系统管理权限。

## 运行环境

- **操作系统**: 兼容Windows、Linux和macOS。
- **依赖**: Python环境，PyQt5库。

## 部署步骤

1. 安装Python环境。
2. 安装PyQt5库。
3. 解压项目文件，运行程序。
